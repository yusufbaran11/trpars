# Contributing to TRPars

## How Can You Help?

### 1. Reporting a Bug
- Open the Issues tab
- Click "New Issue"
- Provide a detailed description

### 2. Ideas/Suggestions
- Issues can be selected as "Development"
- State your ideas

### 3. Code Contribution
- Fork the Repository
- Combine the `feature/feature-name` branch
- Make your changes
- Open a Pull Request

### 4. Documentation
- Write tutorials
- Translate
- You can fix bugs

## Kickstart Edits

Before editing Kickstart:
1. Test for formatting
2. Explain why you are making changes
3. Consider the initial user experience

## Do You Have Questions?

You can sort the discussions!
